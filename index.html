About the Project

The Icebreaker Project is an App designed to help people to get to know each other. Specifically designed for events and meetings Icebreaker is perfect for professional settings, as well as any other kind of event.

About the Creators

The Icebreaker Project is a Capstone project created by a few students from Fairchild Wheeler Interdistrict Magnet Highschool and the source code is available in our Github repositories.
